2. Tahun Kabisat

Algoritma: 

step1 : Tentukan tahun 
step2 : Tentukan hasil
step3 : Hitung Jika Tahun Modulo 4 = 0 
step4 : Jika Tahun Modulo 100 != 0;
step5 : Jika Tahun Modulo 400 =0;
step6 : Hasil adalah "Kabisat"
step7 : Lainya Hasil adalah "bukan kabisat"
step8 : Cetak Hasil
step9 : END.

Pseudocode:

READ Tahun;
STORE Hasil;
CALCULATE IF Tahun modulo  4 = 0 
                      THEN IF Tahun modulo 100 != 0
		Hasil Kabisat 
	                ELSE IF Tahun modulo  400 = 0;
                                 Hasil "KABISAT";
                                 ELSE
                                Hasil "BUKAN KABISAT";
PRINT Tahun adalah Hasil;
END.

3. Laundy Day

Algoritma:

step1 : Pakaian pada mesin cuci 0
step2 : Ketika PakaianFx kurang dari 20
step3 : Lakukan tambah PakaianFx 1
step4 : kembali ke step2 sampai PakaianFx 20
step5 : "mulai mencuci"
step6 : END.

Pseudocode:

READ PakaianFx 1;
WHILE PakaianFx < 20
   DO ADD PakaianFx Plus 1;
PRINT "mulai mencuci";
END.

4. Periksa Kuku

Algoritma: 

step1 : Tentukan nilai awal NomorSiswa sama dengan nol
step2 : Ketika NomorSiswa  kurang dari 40
step3 : Lakukan Syarat jika NomorSiswa dengan kuku Panjang
step4 : Maka NomorSiswa mendapat "hukuman"
step5 : Lainnya NomorSiswa Mendapat "pujian"
step6 : NomorSiswa ditambah 1;
Step7 : Kembali ke step 2 sampai NomorSiswa lebih dari 40 
Step8 : END.

Pseudocode:

GET Siswa 0;
WHILE Siswa < 40;
   DO IF Siswa = Kuku Panjang;
         THEN Siswa "hukuman";
         Else Siswa "pujian";
    ENDIF;
   Siswa + 1;
ENDWHILE;
END.
